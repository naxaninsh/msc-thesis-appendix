
# Thesis Supplementary Materials

This repository is intended to contain supplementary materials associated with a Master's thesis on the visualization of hydrodynamic connectivity.

The repository is currently being prepared for final archival submission. Selected supporting materials will be added after the initial thesis submission. These may include appendix materials, study artifacts, supporting figures, and additional methodological documentation.

For the current submission stage, this repository serves as a stable reference location for future supplementary content.
